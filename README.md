# Pokémon Card Battle Game

A fun **Pokémon card game** built with **React** and **TypeScript**. Players select Pokémon to battle against a CPU opponent. The game uses type advantages (like Rock-Paper-Scissors) to determine the winner of each attack.

---

## Features

- **Player vs CPU** (May Expand to player v player in the future using websocket)
- **Type-based battles**: fire, water, grass, electric, etc.
- **Automatic CPU moves** with random or AI-based decisions
- **Game state tracking**: win, lose, tie, or in progress
- **Reset functionality** to start a new game
- **Responsive UI** with visual type colors

---

## Technology Stack

React (functional components + hooks)
TypeScript for strong typing
CSS / animations for card effects
